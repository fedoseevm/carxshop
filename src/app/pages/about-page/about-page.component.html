The entire project was made using Angular.

  1. Aplikacja jest podzielona na komponenty (components)
  2. Zawiera formularz (form)
  3. Są wykorzystywane serwisy (services)
  4. Dane o samochodach są pobierane z json-serwera, wykonując zapytania Http (Http queries)
  5. Aplikacja zawiera 4 osobne ścieżki (routing)
  6. Używane są dyrektywy strukturalne (*ngFor i *ngIf)
